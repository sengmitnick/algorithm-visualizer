# 计数排序（Counting sort）

计数排序（Counting sort）是一种稳定的线性时间排序算法。该算法于1954年由 Harold H. Seward 提出。计数排序使用一个额外的数组$C$ ，其中第$i$个元素是待排序数组$A$中值等于$i$的元素的个数。然后根据数组$C$ 来将$A$中的元素排到正确的位置。


## 时间空间复杂度

|  |  |
| --- | :--- |
| 平均时间复杂度 | $O(n + k)$ |
| 最坏时间复杂度 | $O(n + k)$ |
| 最优时间复杂度 | $O(n + k)$ |
| 空间复杂度 | $O(n + k)$ |
